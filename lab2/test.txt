Тест-кейс № 1. Удаление аккаунта.
1. Зайти на сайт https://rozetka.com.ua/.
2. Нажать на кнопку личного кабинета "Андрей". 
3. Перейти в раздел "Личные данные".
4. Нажать на кнопку "Удалить аакаунт навсегда".
5. Пометить чекбокс "Я осознанно удаляю свой аккаунт, списки желаний, историю просмотров и заказов. Отказываюсь от возможности получать письма о скидках и ответах на мои комментарии.".
6. Нажать на кнопку "Удалить мой аккаунт".
Ожидаемый результат: Страница перезагружается, появляется всплывающее окно "Ваш аккаунт удалён".

Тест-кейс № 2. Ввод несуществующего промо-кода
1. Зайти на сайт https://rozetka.com.ua/.
2. Перейти в раздел "Ноутбуки и компьютеры".
3. Перейти в раздел "Asus".
4. Выбрать товар (прим. Asus ROG Strix) нажав на него.
5. Нажать на кнопку "Купить". 
6. Нажать на кнопку "Оформить заказ". 
7. Заполнить поля "Имя и фамилия", "Город" и "Мобильный телефон".
8. Нажать на кнопку "Далее". 
9. Нажать на кнопку "Ввести промокод". 
10. Заполнить поле несуществующим промокодом.
11. Нажать на кнопку "Применить"
Ожидаемый результат: Поле для ввода подсвечивается красным цветом.

Тест-кейс № 3. Попытка задействования голосового поиска без подключённого устройства звукозаписи.
1. Зайти на сайт https://rozetka.com.ua/.
2. Нажать на кнопку с изображением микрофона в поле "Поиск".
Ожидаемый результат: Появление всплывающего сообщения с предупреждением.
Полученный результат: Ничего не происходит.

Тест-кейс № 4. Выбор несуществующего населённого пункта для доставки. 
1. Зайти на сайт https://rozetka.com.ua/.
2. Нажать на кнопку с уже выбранным городом (прим. Одесса). 
3. Заполнить поле "Или введите другой населенный пункт Украины" случайным символьным значением (прим. БГТУ). 
Ожидаемый результат: Появляется всплывающее окно с предупреждением "Город не найден. Проверьте написание или введите ближайший к вам!".

Тест-кейс № 5. Попытка ввода телефонного номера с белорусским кодом для активации бонусной системы. 
1. Зайти на сайт https://rozetka.com.ua/.
2. Нажать на кнопку "Бонусы". 
3. Заплолнить поле для ввода телефонным номером содержащим белорусский телефонный код (прим. +375 29 189 53 00). 
4. Нажать на кнопку "Далее".
Ожидаемый результат: Появление всплывающего сообщения с предупреждением или выполнение операции.
Полученный результат: Страница сайта зависает входя в бесконечную обработку запроса.

Тест-кейс № 6. Сменить пароль на пустой. 
1. Зайти на сайт https://my.rozetka.com.ua/.
2. Нажать на кнопку личного кабинета "Андрей".  
3. Нажать на кнопку "Изменить пароль". 
4. Заполнить поле "Введите старый или временный пароль".
5. Оставить поля "Придумайте новый пароль" и "Новый пароль еще раз" пустыми.
6. Нажать на кнопку "Сохранить".
Ожидаемый результат: Смена пароля не происходит, поля "Придумайте новый пароль" и "Новый пароль еще раз" подсвечиваются красным цветом.

Тест-кейс № 7. Задать пароль не отвечающий требованиям политики безопасности сайта.
1. Зайти на сайт https://my.rozetka.com.ua/.
2. Нажать на кнопку "войдите в личный кабинет". 
3. Нажать на кнопку "Зарегестрироваться". 
4. Заполнить поля "Ваше имя" и "Эл. почта или телефон".
5. Запололнить поле "Придумайте пароль" цифровой комбинацией "123".
6. Нажать на кнопку "Зарегестрироваться".
Ожидаемый результат: Регистрация не проходит, поле "Придумайте пароль" подсвечивается красным цветом.

Тест-кейс № 9. Попытка создать "Список желаний" без ввода названия.
1. Зайти на сайт https://my.rozetka.com.ua/.
2. Нажать на кнопку личного кабинета "Андрей".
3. Перейти в раздел "Списки желаний".
4. Нажать кнопку "+ Новый список".
5. Оставить поле "Мой список желаний" пустым. 
6. Нажать кнопку "Сохранить".
Ожидаемый результат: Создаётся список с именем по умолчанию "Мой список желаний".

Тест-кейс № 10. Попытка создания два "Списка желаний" с одинаковыми названиями.
1. Зайти на сайт https://my.rozetka.com.ua/.
2. Нажать на кнопку личного кабинета "Андрей".
3. Перейти в раздел "Списки желаний".
4. Нажать кнопку "+ Новый список".
5. Заполнить поле "Мой список желаний" символьным значением "Список 1". 
6. Нажать кнопку "Сохранить".
7. Нажать кнопку "+ Новый список".
8. Заполнить поле "Мой список желаний" символьным значением "Список 1". 
9. Нажать кнопку "Сохранить".
Ожидаемый результат: Создаётся два списка с одинаковыми названиями.
